## Key Patterns

**Anti-Patterns (Common Wrong Answers):**
- Parsing natural language for termination
- Sentiment-based escalation
- Self-reported confidence for decisions
- Prompt-based enforcement for critical rules
- Generic error messages
- Same-session self-review
- Too many tools (18+)

**Correct Patterns:**
- Check stop_reason for termination
- Explicit business rules for escalation
- Programmatic hooks for critical enforcement
- Structured errors with context
- Fresh session for unbiased review
- 4-5 tools per agent optimal
- Explicit context passing in multi-agent

For detailed explanations, refer to the study guide sections corresponding to each domain.
